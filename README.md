# 摸鱼高手

一个浏览器即开即玩的躲避追击小游戏，无需下载和安装。

## 运行方式

直接用浏览器打开 `index.html` 即可开始。

## 操作说明

- `W` `A` `S` `D`：控制大鱼移动
- `Space`：在持有炸弹时清空全屏老板

## 玩法规则

- 老板会从屏幕四周不断刷新并追击你，时间越久越危险。
- 道具说明：
  - 咖啡：提升移动速度（持续一段时间）
  - 小鱼：立即加分
  - 四角星：无敌 3 秒
  - 炸弹：每 10 秒尝试刷新，拾取后按空格触发
- 被老板碰到会结束游戏，可使用一次“看广告复活”。
- 最高分会自动保存在浏览器本地存储中。
